## 二、 全局核心规则说明

### 1. 热区与箭头交互规则

* **普通物品（未交互/模糊态）**：显示 **黑色普通箭头**（`arrow_black.png`），✅ 可点击。
* **物品清晰化/解开后**：黑色箭头永久消失，❌ 不可再次点击。
* 
**氛围交互点（如水龙头、书架书籍）**：点击后直接弹底部文本框，无放大层，交互一次后 **黑色箭头永久消失** 。


* 
**空间切换/出入口通道**：始终显示 **白色专用箭头**（`arrow_white_exit.png` / `arrow_left.png` / `arrow_right.png`） ，始终保持激活状态，✅ 允许玩家自由往返于已解锁的房间 。



### 2. 阿尔茨海默失语机制（基于实时混沌值 `chaosLevel`）

* 
**阶段一（0-3分钟，chaosLevel <= 30）**：系统稳定，点击选项清晰明确，正确辨认触发短暂的完整回忆 。


* 
**阶段二（3-7分钟，chaosLevel 31-70）**：语言系统解构 。选项文字转换为带有刚体物理效果的粒子 ，**鼠标靠近时会像尘埃一样被吹散/向外弹开** 。文字内容在“真实”与“遗忘的错觉”之间高频闪烁 。


* 
**阶段三（7-10分钟，chaosLevel > 70）**：终极崩溃 。文本脱离控制全屏无序做布朗运动 ，通过 `innerHTML` 机制自动高频、死循环、无序地重叠堆叠（Clone & Overlap）输出崩塌的独白 ，剥夺控制权 。


* ⚠️ **【特殊机制修正】**：在书房及卧室床头柜的阶段三，即使文字在全屏做无序布朗运动、视觉疯狂堆叠，**只要玩家能高难度地精准捕捉并成功点击到飘散中的正确选项，系统依旧会激活 click 事件，允许触发对应的回忆！**

## 四、 结局演出详细流程

### 结局一：真结局 ——「记忆余烬」 (True End)

* **触发条件**：在镜子前正确猜出主角名称（`isTrueEndRoute == true`）。
* **演出分解步骤**：
1. **场景重定向**：点击客厅大门白色箭头后，网页画布平滑淡出，载入真结局专属大门打开底图（`door_open_true.png`）。
2. **光效粒子渲染**：通过 CSS/Canvas 动画，从图片中“门”的坐标中心点开始，向外发散出柔和的**白色扩散光效（White Flare Effect）**。光效在 3 秒内逐渐扩大，最终将整个网页画面完全吞噬、渐变为**纯白色**。
3. **叙事对白涌现**：画面变为纯白后，屏幕中央载入 UI 对话框（`dialog_bg.png`），以流式手写体风格输出主人公与女儿小雪的历史重逢原音对白：
* 💬 **对白文本**：
> “妈妈，你怎么一个人走到这里来啦？手怎么这么冰……”
> “小雪……妈妈今天，好像把你的名字记住了……”




4. **自动退出/结束**：对白播放完毕 **5 秒后**，画面平滑淡入黑底滚动演职员表（Credits），游戏自动退出至主菜单。



---

### 结局二：坏结局 ——「门外的色彩」 (Bad End)

* **触发条件**：在镜子前未能猜出正确名字（`isTrueEndRoute == false`）。
* **演出分解步骤**：
1. **第一阶段（大门开启）**：点击客厅大门白色箭头后，网页画布剧烈震颤（Screen Shake）并平滑淡出，载入坏结局专属的大门打开冰冷写实特写底图（`door_open_bad.png`）。
2. **第二阶段（认知解构与双图闪回）**：系统通过 JavaScript 定时器（`setInterval`）启动高频交替动画。彩色高饱和度扭曲背景（`bad_chroma_distorted.png`）与黑白高对比度扭曲背景（`bad_mono_distorted.png`）在 **【100ms - 250ms】** 之间进行无序随机的高频交替切换闪回。配合 CSS filter 产生画面数字撕裂卡顿（Glitch Effect），营造极端的精神撕裂氛围。
* 💬 底部对话框（`dialog_bg.png`）手写体文字高频闪烁，输出主角以为获得自由的错觉：
> “门开了……我终于逃出来了。外面好热闹，到处都是颜色……”
> “我自由了。我是晓雪，我要去上学了……”


* ⏳ **持续时间**：此高频交替闪回效果与第一阶段独白持续播放 **6 秒**。


3. **第三阶段（残酷现实与标题定格）**：定时器关闭（`clearInterval`），扭曲闪回骤停！整个网页画面瞬间失去所有画面色彩，强行抹除一切背景，转为纯黑画布（`#000000`）。
* 📷 **全屏标题定格**：纯黑底图正中央平滑渲染出带有粗糙颗粒感、如同灰烬消散的白色美术大字标题底图 —— **《门外的色彩》**（`bad_title_screen.png`）。
* 💬 **系统播报文本**：此时底部对话框样式改变，转换为如同报纸新闻播报的冰冷宋体字，输出社会新闻：
> “【寻人启事】陈慧君，女，78岁，患有严重阿尔茨海默症。于今日下午从家中走失，走失时身穿灰色针织衫……如有知情者，请速与家人联系。她的女儿一直在等她回家。”


* ⏳ **自动退出机制**：该结局标题与新闻启事在屏幕常驻显示 **5 秒钟** 后，底部对话框消隐，画面彻底淡出入终局，游戏通关结束。

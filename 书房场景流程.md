## 3. 场景美术与声音说明

> 📷 **视觉图层说明**：全盘采用高对比度、泛黄且带有强烈纸张霉味颗粒感的 **20世纪70年代真实上海民居书房老照片** 作为网页全屏背景（图片资产：`./assets/scenes/study/study_70s.png`）。画面中包含堆满手写病历本的老旧木书桌、插满旧书与小人书的木书架。

### 声音系统设计（基于声音物质性的粒子化粘稠）

* **声轨一：秩序之声（0 - 3 分钟 / chaosLevel 0 - 30）**：缓慢沉闷的书页翻动声与窗外偶尔传来的自行车铃声。声音定位明确，保持极简清晰的具象特质。
* **声轨二：粒子化粘稠拉伸（3 - 7 分钟 / chaosLevel 31 - 70）**：利用 JavaScript Web Audio API 的粒子合成特效，将翻书的声音**物理拉长 200 倍**，异化为如同废旧工厂钢铁摩擦般的刺耳低频轰鸣（Drone）。
* **声轨三：历史原音涌现（7 - 10 分钟 / chaosLevel 71 - 100）**：多声道叠加倒放“医生确诊阿尔茨海默症”的诊断录音片段（“……陈慧君女士，您需要每天吃药……”），回音高频重叠交织。

## 4. 核心物件交互流程解构

### 物件一：书桌上的病历单与药瓶 (The Medical Record & Medicine)

* **属性/状态**：核心叙事真相揭谜点（模糊态 ➔ 语义辨认 ➔ 清晰回忆释放）
* **历史资产照片**：`./assets/items/study/medical_record.png`（70年代上海第一人民医院的纸质病历本与“多奈哌齐”小药瓶特写照片）
* **热区图片**：`./assets/ui/arrow_black.png`

#### 动态交互流程（含chaosLevel>70捕捉判定机制）：

```text
【第一层】书房场景主视图
    │
    玩家点击书桌上病历本位置的黑色热区箭头
    ↓
【第二层】物品放大悬浮窗 (<div id="overlay">)
    │
    ├── 渲染病历单与小药瓶的放大历史老照片：medical_record.png
    └── 动态加载交互文本区域 (<div id="interaction-text">)
            │
            ├── IF chaosLevel <= 30 (阶段一：秩序)
            │     ├── 显示清晰的手写体选项：[A. 病历本与药瓶] [B. 日记本与糖罐] [C. 记账本与香水]
            │     └── 玩家点击正确选项 [A] ➔ 猜名正确！
            │           ├── 触发【回忆四：确诊静态切片（memory4_diagnosis.png）】
            │           ├── 药瓶标签从误认的“很甜的糖”溶解转换为真实的“盐酸多奈哌齐”
            │           └── 遮罩层关闭，该位置的黑色热区箭头永久消失，计入真结局进度。
            │
            ├── IF chaosLevel 31-70 (阶段二：瓦解)
            │     ├── 文本选项拆解为 <span> 字符粒子的物理错位态
            │     ├── 玩家鼠标靠近 [A] 时，文字受算法驱动在“病历”与“糖果”之间高频闪烁并向外弹开
            │     └── 玩家必须精准捕捉并完成点击 [A] ➔ 顺利触发确诊回忆（逻辑同上，箭头永久消失）。
            │
            └── IF chaosLevel > 70 (阶段三：终极涌现)
                  ├── 【视觉与文本干扰逻辑】：
                  │     文字完全脱离按钮框，在全屏做密集的无序布朗运动。同时系统利用 innerHTML 
                  │     漏洞式机制疯狂、高频、成百上千次地在最前端堆叠输出黑白色、不同字号的内心独白：
                  │     “海马体……严重萎缩……不治之症……陈慧君女士……不要忘记吃药……”
                  ├── 【底层核心交互保留】：
                  │     **虽然整个屏幕被文本堆叠吞噬、选项在疯狂飘散移动，但该正确选项 [A] 按钮的 
                  │     click 事件监听器（EventListener）依旧处于激活状态！**
                  │     如果玩家顶着巨大的视觉干扰和移动速度，成功“捕捉”并点击到了飘散中的正确选项 [A]：
                  │     ➔ **奇迹般中断字体的狂暴涌现，强制拉起【回忆四：确诊静态切片】，计入真结局进度！**
                  └── 【失败惩罚】：如果玩家误点其余选项或直到倒计时结束也未能捕捉到 [A] 
                        ➔ 遮罩层自动崩溃，热区永久消失，该段核心真相记忆宣告永久丢失。

```
